SkillBridge API is a production backend connecting students with expert tutors. Authentication, scheduling, bookings, and reviews — all in one RESTful service.

View Live API ↗ GitHub Database Schema

What it does

  • auth Email/password and Google OAuth via Better Auth. Email verification, JWT sessions, and role-based access control for students, tutors, and admins.
  • tutor profiles Rich tutor listings with bio, hourly rate, categories, languages, education, experience, and manual verification status.
  • scheduling Weekly recurring availability slots with real-time booking, status lifecycle management, and meeting link delivery.
  • reviews 5-star review system tied to completed bookings. Aggregated ratings update the tutor profile automatically.
  • search Filtered tutor discovery by subject, language, rating, experience, and availability — with pagination and sort options.
